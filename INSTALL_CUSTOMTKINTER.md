# Install CustomTkinter

To run the modernized Cubase/Nuendo Duration Calculator, you need to install CustomTkinter:

## Quick Installation:

```bash
pip3 install customtkinter
```

## If you get permission errors:

```bash
pip3 install --user customtkinter
```

## To upgrade if already installed:

```bash
pip3 install customtkinter --upgrade
```

## What CustomTkinter Provides:

✅ **Beautiful rounded buttons** instead of ugly gray rectangles
✅ **Native macOS dark mode support** - automatically adapts to system appearance
✅ **Modern styling** with proper colors, fonts, and hover effects
✅ **Professional appearance** that matches modern macOS apps

## After Installation:

Simply run the app as usual:
```bash
python3 "Multiple Event Duration Calculator.py"
```

The app will now have gorgeous, modern buttons with proper sizing and styling!
